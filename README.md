# Redis-lens

Rust-based terminal user interface (TUI) application for browsing Redis and Valkey databases. It's designed as a simple, interactive tool to explore keys and their values in a Redis instance.

## Features

- Browse Redis keys in an interactive list
- View key values, types, and TTL information
- Connect to Redis servers with configurable URL and database
- Command-line options for specific operations
- Mock client for testing without a real Redis connection
